### linux 文件权限
增加运行权限（如.sh）：`chmod +x <文件名>`，`chmod +x *.sh`

` chmod [用户范围][操作符][权限符号] <文件名>`

+ 用户范围 包括：u (user/owner), g (group), o (others), a (all)。**可省略，默认是a**

+ 操作符 包括：+ (添加), - (移除), = (赋值)。

+ 权限符号 包括：r (读), w (写), x (执行)。

#### 权限字符串的拆解
以`r--r------`为例子，首先拆成三组`r--`,`r--`,`---`

* **r (read)**：读取权限，分值为 **4**
* **w (write)**：写入权限，分值为 **2**
* **x (execute)**：执行权限，分值为 **1**
* **- (none)**：无权限，分值为 **0**
  
于是
| 用户类别 | 权限字符 | 计算公式 | 八进制值 |
| --- | --- | --- | --- |
| **拥有者 (User)** | `r--` | $4 + 0 + 0$ | **4** |
| **所属组 (Group)** | `r--` | $4 + 0 + 0$ | **4** |
| **其他人 (Others)** | `---` | $0 + 0 + 0$ | **0** |

或者我们可以采用符号表达
```bash
chmod u=r,g=r,o= stderr.txt
```
* `u=r`：拥有者仅设为只读。
* `g=r`：所属组仅设为只读。
* `o=`：其他人权限全部清空（等同于 `---`）。

## C语言的编译过程
+ **预处理**：主要处理源代码文件中以 # 开始的预处理指令
+ **编译**：产生汇编代码 `gcc -S hello.i -o hello.s`
+ **汇编**：汇编代码转换为机器代码，此时生成的`.o`文件即为**目标文件**
+ **链接**：把多个目标文件的代码段放在一起、数据段放在一起，然后和一些写好的库函数（比如 scanf 和 printf ）放在一块共同形成可执行文件。

## C语言的变量存储类别
**标识符--一片连续内存区域的首位。**
+ **分清两种工具——编译器和链接器**
### 变量作用域--编译器
+ block scope: 花括号内的代码区域，此外还包括函数的形式参数声明
+ file scope: 函数外部的区域，如全局变量
### 变量存储期--编译器
+ `static`：程序执行一直存在，内存空间大小和起始地址不变
+ `auto`自动存储期（可省略）：**变量作用域**结束后就释放，且如果不初始化值不确定，
### 链接--链接器
描述是否可以在别的文件中使用这个变量
+ 内部链接：只能在定义他的本文件使用，如所有static的全局变量
+ 外部链接：所有文件使用，全局变量默认
+ 无链接：没有属性，函数内变量是无链接的（包括声明）

### 两种变量声明的方式
+ 定义式声明：创建对象，如`int a;`
+ 引用式声明：不会创建对象，假设定义在其他地方，如`extern int a;`
  
### 函数的存储类别
函数只有链接属性，外部和内部（需要加`extern`）
```C
#include <stdio.h>
/* main.c */
static int v;

extern int func(int value);

int main() {
    int value = 1; //局部变量
    v = func(value); //x=2, y=2 -- v = 4
    {
        extern int value; //链接的value.c中的value = 2
        v += func(value);
    }
    printf("%d\n", v);
    return 0;
}
```
```C
/* func.c */
int func(int value) {
    static int x = 1; //静态局部变量，main.c中初始化一次后不会销毁
    int y = 1; //自动局部变量，每次调用函数都会重新创建
    x += value;
    y += value;
    return x + y;
}
```
```C
/* value.c */
int value = 2;
```

## 预处理指令
### 函数式宏定义
+ 发生在预处理阶段，只发生**形式替换**
+ 定义函数式宏的时候要格外小心，尤其要**多加括号**
+ 避免进行副作用操作(比如自增自减)
例子：`#define MAX(a,b) ((a)>(b)? (a) : (b))`
#### 多语句的宏定义
```C
#define MACRO_NAME(para1, para2) \
    do { \
        express1; \
        express2; \
    } while(0)
```
加上do while是为了保持macro后有分号符合编程习惯同时，不影响上下文。
## `#`运算符
+ 自动把**传入的参数**用双引号括起来称为**一个字符串**，并且实参中的连续多个空格会被替换成一个空格。(所以它不需要被双引号扩着)
```C
#define SQ(x) printf("the square of "#x" is %d", ((x)*(x)))
```
+ 注：紧挨的字符串会自动合并哦！
+ `##` 把前后两个预处理符号连接成一个预处理符号

### 条件预处理
```C
#ifndef _STDIO_H
#define _STDIO_H        1
...
#endif
```
如果之前未定义这个宏，则把中间的部分包含在预处理结果中，否则不出现
+ 防止重定义

### 与typedef的区别
 typedef 一般用来定义类型的别名，定义与平台无关的数据类型，
 + 与struct结合使用,能够保证定义的所有变量均为同一类型，而 #define 则无法保证。
