# 磁盘管理

## 基本概念
+ 扇区
+ 磁道：一个盘片上的不同半径的同心圆
【磁道+扇区 定位块】
+ 柱面：不同盘片相同半径的磁道所组成的柱面
【磁头组：这些磁头只能读不同盘片的同一个柱面】

**每个磁盘有两个面，每个面都有一个磁头(head)**

![alt text](assets/image-32.png)

## 磁盘的组织
信息 映射为 一个三维坐标
定位信息：柱面+磁头+扇区

## 磁盘访问时间
**寻道时间**： 启动时间s，移动n个磁道
Ts＝m × n + s

**旋转时间**：
Tr = 1/(2r) r:转速 RPM

**传输时间**：
读写字节数b 旋转速度r 磁道上的字节数N
- 单位时间内转r圈，总共读取rN个字节；需要读取的总字节数为b
所以 Tt =  b/(rN)

**总延迟时间**
Ta = Ts ＋1/(2r)＋ b/(rN)

判断主要矛盾，具体情况具体分析

### 扫描算法（SCAN）电梯调度

+ 发出请求后，磁头**按照一个方向移动**，移动过程中**遇到了请求进行服务**
+ 服务后**判断**该方向上是否还有请求，有则继续移动，**无则改变方向**

问题：摆动式的扫描方法，两侧磁道被访问的频率降低
