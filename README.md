# 操作系统 (2026 春)

BUAA 操作系统课程学习仓库，包含课程笔记、实验报告、理论作业及考试复习资料。

## 目录结构

```
.
├── Notes/          # 课程笔记（按章节编号）
├── lecture/        # 课堂讲稿整理与PPT识别提取内容
├── Slides/         # 课程课件 (PDF)
├── lab/            # 实验报告 (lab0 ~ lab6)
│   └── pre/        # 预习笔记与补充材料
├── Instruction/    # 实验指导书与小测
├── 理论作业/       # 理论作业 (作业1 ~ 作业7)
├── mid-term/       # 期中考试复习
└── final-exam/     # 期末历年真题与复习
```
